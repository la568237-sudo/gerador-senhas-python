# Gerador de Senhas em Python

Projeto simples que cria senhas aleatórias usando Python.

Fiz esse projeto para praticar lógica de programação e uso de bibliotecas.

## O que o programa faz

- gera uma senha aleatória
- mistura letras, números e símbolos
- o usuário escolhe o tamanho da senha

## Tecnologias

Python

## O que aprendi

- uso da biblioteca random
- manipulação de strings
- criação de funções

## Melhorias futuras

- opção de copiar senha
- escolher tipos de caracteres
- criar interface gráfica
